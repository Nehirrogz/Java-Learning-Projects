# ATM Console Application
This project is a simple ATM simulation developed to practice **loops**, **conditional statements**, and **user input** in Java.

## Features

The application allows the user to:
* **Check Balance:** View current account balance.
* **Deposit Money:** Add funds to the account.
* **Withdraw Money:** Withdraw funds (with balance check).
* **Exit:** Terminates the session safely.

## Tech Stack & Concepts

* **Language:** Java (JDK 21)
* **IDE:** Apache NetBeans
* **Concepts:**
    * `do-while` Loop (For the menu cycle)
    * `switch-case` (For menu selection)
    * `if-else` (For balance validation)

## How to Run

1. Clone the repository or download the source code.
2. Open `AtmProject.java` in your IDE (IntelliJ, NetBeans, Eclipse).
3. Run the file and follow the console instructions.
